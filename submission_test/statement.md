# A + B Problem

## Description

Given two integers A and B, output their sum.

## Input

The first line contains an integer T, the number of test cases.
Each test case consists of a single line containing two integers A and B.

## Output

For each test case, output the sum of A and B on a new line.

## Constraints

- 1 <= T <= 100
- 0 <= A, B <= 10^18
